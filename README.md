# desafioalura
Projeto de encriptador proposto pela alura no curso de Iniciante em Programação T7 - ONE
o projeto faz a encriptação do texto digitado pelo usuario e depois é possivel fazer a desincriptaçao.
O projeto que desenvolvi com a ajuda da alura cria uma chave de encriptaçao em que na Funcao principal do projeto faço um loop, em que vai iterar com cada elemento da lista,
que vai aumentando em 1 a cada iteraçao passando por cada chave e depois faz a troca atraves do replaceAll, e desincripta fazendo o processo reverso.
